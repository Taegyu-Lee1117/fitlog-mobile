# FitLog

FitLog는 식사 칼로리와 운동 시간을 빠르게 기록하고 오늘의 목표 진행률을 확인하는 오픈소스 모바일 앱입니다. 회원가입이나 서버 없이 기록을 휴대폰에 저장합니다.

## 주요 기능

- 식단 이름과 섭취 칼로리 기록
- 운동 이름과 운동 시간 기록
- 하루 목표 진행률 표시
- 기록을 길게 눌러 삭제
- Android와 iOS 지원

## 실행 방법

```bash
npm install
npm start
```

휴대폰에 Expo Go를 설치하고 QR 코드를 스캔합니다.

## 테스트

```bash
npm test
```

## 프로젝트 방향

사용자 설정 목표, 주간 통계, 데이터 내보내기와 접근성 개선을 추가할 예정입니다. 제안과 기여를 환영합니다.

## 개인정보와 건강 안내

현재 기록은 사용자의 기기에만 저장됩니다. FitLog는 의료 서비스가 아니며 건강 상태를 진단하지 않습니다.
