# OpenAI 오픈소스 지원 프로그램 신청 초안

아래 문구는 프로젝트를 실제로 공개하고 운영한 뒤 사실에 맞게 수정해 사용하세요. 아직 없는 사용자 수, Star, 다운로드 수나 성과는 작성하지 마세요.

## 담당 역할

일상에서 운동과 식단을 꾸준히 기록하려 할 때 복잡한 가입 절차와 과도한 기능이 기록 습관을 방해한다는 문제에서 FitLog를 시작했습니다. 프로젝트 기획부터 화면 구성, 기록·합산 로직, 기기 내 데이터 저장, 테스트와 문서화를 담당하고 있습니다. Android와 iOS에서 누구나 간단히 사용할 수 있는 오픈소스 앱을 목표로 유지보수하며, 사용자 의견과 외부 기여를 바탕으로 목표 설정, 주간 통계와 데이터 내보내기 기능을 발전시킬 계획입니다.

## 프로젝트가 프로그램에 적합한 이유

FitLog는 식단 칼로리와 운동 시간을 빠르게 기록하는 오픈소스 모바일 앱입니다. 회원가입이나 외부 서버 없이 기기 안에 데이터를 저장해 진입 장벽과 개인정보 부담을 낮췄습니다. Android와 iOS를 함께 지원하며, 핵심 기록 로직의 자동 테스트와 기여 가이드를 제공해 누구나 개선에 참여할 수 있습니다. 향후 실제 사용자 피드백을 공개 이슈로 관리하고 접근성, 통계와 데이터 이동성을 강화하겠습니다.

## API 크레딧 사용 계획

API 크레딧은 FitLog의 선택형 AI 기능을 실험하고 평가하는 데 사용할 계획입니다. 사용자가 직접 입력한 식사 설명을 구조화된 기록 초안으로 변환하고, 일주일 기록을 바탕으로 비의료적 습관 요약을 제공하는 기능을 개발하겠습니다. 개인정보 최소 수집, 명시적 동의와 비용 제한을 우선하며, 프롬프트·평가 기준·샘플 데이터와 구현 과정을 저장소에 공개해 다른 개발자도 안전한 건강 기록 기능을 재현하고 개선할 수 있도록 하겠습니다.

## 추가로 알리고 싶은 내용

FitLog는 지원 신청만을 위한 일회성 저장소가 아니라, 직접 사용하면서 개선할 장기 프로젝트로 시작했습니다. 작은 기록 경험을 단순하게 만드는 데 집중하고 있으며 테스트, 개인정보 안내, 기여 문서를 첫 버전부터 포함했습니다. 앞으로 공개 로드맵과 이슈를 통해 사용자 의견을 반영하고 초보 기여자가 참여하기 쉬운 작업을 정리하겠습니다. Codex를 활용한 개발 과정을 문서화해 비전공자와 초기 개발자도 오픈소스 모바일 앱을 유지하는 방법을 배울 수 있게 하겠습니다.
